How it Works

  1. Rent the kit (pay £50 + refundable £175 deposit)
  2. Plug in USB mic → browse to measurely.local → hit “Measure”
  3. Read Dave’s 3 plain-English tips
  4. Move stuff, re-measure, hear the difference
  5. Re-pack, stick on pre-paid label, drop at post-office
  6. We refund deposit within 24 h of safe return
