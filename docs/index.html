How It Works

  1. Rent the kit (£50 + refundable £175 deposit)
  2. Plug & Play: Plug in mic → go to measurely.local
  3. Measure: Press the button and let Dave analyze the sweep.
  4. Fix: Read Dave’s 3 plain-English nudges (e.g. "Move sofa 10cm left").
  5. Return: Re-pack and drop at the Post Office using the prepaid label.
